## Run Locally

Clone the project

```bash
  git clone https://github.com/KobyEthan/sorting-visualizer.git sorting-visualizer
```

Go to the project directory

```bash
  cd sorting-visualizer
```

Install dependencies

```bash
  npm install
```

```bash
  npm start
```
